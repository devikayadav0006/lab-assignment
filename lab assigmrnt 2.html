Software

    Software consists of computer programs that instruct the execution of a computer.[1] Software also includes design documents and specifications.

  • operating systems:which manage hardware resources and provide services for applications
  • application software:which performs specific tasks for users
  • Placement:refers to the process of getting a job in the software industry, which can be facilitated by a university's campus placement program or through specific job placement services
  • Company Name:TEKsystems,Software Placement Group (SPG),ABC Consultants ect.
History

Main History of software
The integrated circuit is an essential invention to produce modern software systems.[2] The first use of the word software to describe computer programs is credited to mathematician John Wilder Tukey in 1958.[3][4] The first programmable computers, which appeared at the end of the 1940s,[5] were programmed in machine language.
