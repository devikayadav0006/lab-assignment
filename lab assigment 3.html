Function of image tag

The img tag used to embed an image in an HTML page. Images are not technically inserted into a web page; images are linked to web pages. The tag creates a holding space for the referenced image. tag

Function and uses of the Tag:-

  • Displays images like logos, photos, icons, or illustrations
  • Enhances visual appeal and user experience
  • Can be styled and positioned using CSS
Syntax with Key Attributes
Explanation of Attributes
Attribute Purpose src Source of the image file. This can be a local path (images/sunset.jpg) or a URL (https://example.com/sunset.jpg). In VS Code, it helps you link to your image assets alt Alternative text shown if the image fails to load. Also improves accessibility for screen readers. VS Code may highlight missing alt as a warning. width & height Define the dimensions of the image in pixels. Helps control layout and responsiveness. longdesc Specifies a URL to a detailed description of an image srcset Specifies a list of image files to use in different situations
