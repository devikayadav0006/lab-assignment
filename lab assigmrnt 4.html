Lists Type in HTML

Differences between ol and ul list


Key Differences:-
Tag Full Form Purpose List Style
ol Ordered List Displays items in a specific order Numbers or letters
ul Unordered List Displays items without order Bullets or icons


Simple code for the Ordered List
  1. Wake Up
  2. Ready for college
  3. Attend all the sessions
  4. Had Lunch
  5. Come back from college
  6. Own time for studies

Uses case:used to create an HTML ordered list for items where sequence is important, such as step-by-step instructions, rankings, or procedures. It automatically numbers the list items and can be customized with Roman numerals or letters using the type attribute


Simple code for the Unordered List
  • Tea
  • pink
  • red
  • blue

Use case:used in HTML to create an unordered list, which is a collection of items without a specific order, typically displayed with bullet points. It is used to group related items where the order is not important, such as product features, navigation menus, or grocery lists.
